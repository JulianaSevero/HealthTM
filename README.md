# HealthTM

Esse repositório contém templates e artefatos de execução do HealthTM, um framework de threat modeling para ambientes de saúde proposto como dissertação do curso de informática biomédica. Cada template contém comentários que ao editar, aparecem para auxiliar na contextualização para preencher o documento.

//
